# Controle Finanças

## Este aplicativo foi feito na intenção de facilitar o controle da vida financeira dos usuários. Ele tem como objetivo registrar todos os gastos e lucros do usuário, proporcionando uma visualização facilitada de como está sua vida financeira e separar suas contas de forma categorizada.

### Tela de Introdução

#### A primeira do aplicativo é uma tela de introdução, onde o usuário possui algumas maneiras de se autenticar no aplicativo.

![Tela de Introdução](https://github.com/WilliamDCGomes/controle-financas/blob/main/ReadmeFile/Tela%20de%20Introdu%C3%A7%C3%A3o.jpg)

### Tela de Login

#### Logo em seguida temos a tela de Login, onde o usuário irá se autenticar no aplicativo.

![Tela de Login](https://github.com/WilliamDCGomes/controle-financas/blob/main/ReadmeFile/Tela%20de%20Login.jpg)

### Tela de Cadastro

#### Caso seja o primeiro acesso, o usuário pode solicitar um cadastro, conseguindo assim logar no aplicativo.

![Tela de Cadastro](https://github.com/WilliamDCGomes/controle-financas/blob/main/ReadmeFile/Tela%20de%20Cadastro.jpg)

### Redefinir Senha

#### Existe também a possibilidade do usuário redefinir a sua senha, caso o mesmo não se lembre da senha cadastrada.

![Redefinir Senha](https://github.com/WilliamDCGomes/controle-financas/blob/main/ReadmeFile/Popup%20de%20Redefinir%20Senha.jpg)

### Dashboard

#### Uma vez logado no sistema, o usuário tem acesso à um Dashboard, mostrando algumas informações sobre suas contas, com algumas ações rápidas.

![Dashboard](https://github.com/WilliamDCGomes/controle-financas/blob/main/ReadmeFile/Dashboard.jpg)

### Contas Separadas por Categorias

#### Logo a baixo, é mostrado várias listas de contas, separadas por grupos a fim de facilitar a visualização das contas pendentes ou pagas.

![Contas Separadas por Categorias](https://github.com/WilliamDCGomes/controle-financas/blob/main/ReadmeFile/Categorias%20de%20Contas.jpg)

### Detalhes de Conta

#### Caso você selecione alguma das contas, você será direcionado para uma tela onde mostrará os detalhes dela. A tela irá informar alguns detalhes como data de vencimento, valor e o status atual. Caso a conta já tenha sido debitada, um simbolo amarelo irá aparecer no canto superior direito tela indicando isso, mas caso a conta esteja pendente, será habilitado um botão para marcar a conta como paga.
